持ち出せるIP電話、050plus は MVNO と Simフリー端末を加速するか

スマートフォンでの音声・ビデオ通話/チャット のアプリが熱い

去年しらべた時以降に、LINE(ライン) がテレビCMまで打ってブイブイいわせているみたいですね。兎に角この手のアプリは相手あってのモノなんで、Facebook とかに完全にシェアを奪われてしまう前の今が勝負所なんでしょう。

ただ若い人は兎も角として、やはり暫くは電話番号つかって電話をかけるというのは、まだまだ主流で在り続けると思います。

なので去年はIDEOSで SkypeIn / Skuypeout を試す などしましたが、通話品質は兎も角として、割り切れば使えなくもないな、と言った印象でした。ただ、番号通知や不在時の挙動など、端末との統合っぷりがイマイチだった事もあり、それをメインにしようとは思えませんでした。

 

 

持ち出せるIP電話、050plus

そうこうしている内に、050 plusが、iPhoneのみならずいつのまにかWindowsやAndroidでも使える様になっています。

050plusとは？を要約するとこんなところ。

  • NTTコミュニケーションズ のIP電話サービスが利用できるiPhone / Android / Windows アプリ
  • 月額基本料315円 (Skypeinとだいたい同じ)
  • 050番号がもらえる (Skypeinと同じ）
  • 支払いはクレジットカード
  • 通話品質は電波状態に依存。WiFi直下なら快適
  • 登録時に携帯電話番号認証さえクリアすれば、データ通信のみのAndroidとか極端な話iPod / iPadでも可

誰でも最初に考えるのは、日本通信の b-mobile イオン専用Simで運用すれば、スマホが980円/月で使えるジャン！ですが、そのへんはb-mobile も危険を察知したらしく、通信を制限している様で使い物にならないようです。（b-mobile イオン専用SIMで音声通話のサービスを始めてはりますし）

日本通信なら、b-mobile Fair（120日間または1GBまで 9,800円）か、b-mobile 1GB定額 (3,100円/月)ならイケるとの報告あり。DoCoMoのMVNOなんでエリアは問題ないでしょう。

あとSo-net モバイル3G (これもDoCoMoのMVNO)でもイケるらしいです。

ドコモのパケット定額や、各社MVNOなど、VoIPを制限しているもの多数だから注意が必要です。状況は変化するので、ググッて調べましょう。

 

 

登録が簡単/判りやすいのがイイ

登録はとっても簡単、必要なものはクレジットカード番号と、携帯電話（自動音声で本人確認がある）。

ウィザードにしたがって入力すれば、10分もかからない。

入会時には、友達紹介コードを入れましょう。

紹介したひと、された人双方に、500円分のQuoカードが抽選で当たるそうです。

参考サイトにいっぱい書いてますし、私のんも書いておきます。是非。

489GPA52

紹介相手には伝わらないようですので、気兼ねなく。

マニアな方々は、スマホ に SIPクライアントをぶっこんで どこでも IP電話を使える様にする御仁もおられるようですが、050plus はネームバリューにものを言わせ、それを安心・簡単にしてしまったのです。

この安心・簡単って、普及にはすごく大切な要素なんですよね。いくらSkypeでほぼ同じ事ができるとしても、面倒なんですよね。登録にPCが要るし、英語のページとか出てきた時点でアウトです。

 

 

050plus と Simフリー端末

個人的には、この050plus は、Simフリー端末でこそ威力を発揮するものだと思っています。

2012年以降はいよいよガラケーからスマホへのシフトが加速しそうですが、たかだか携帯電話ごときに月々6000円前後も払えるかい！と言う方が大半では無いでしょうか。

そんな人には、SimフリーのスマホをAmazonとかで購入して、DoCoMoの音声のみのプランで契約した携帯のSimを刺す（800円位/月）、というのを提案する様にしています。メールはGMailを使えばよいし、家と出先にWiFiがあればなんとかなります。急ぎの用事は音声でもSMSは使えるんで。各社乗り入れ可能になったいま、SMSってこういう場合にはアリかと。

いや外でもGMailとかネット見たいねん、と言う方には、これまたSimフリーのスマホにデータ通信プランのSimカードを指して、ガラケー（音声のみ契約）との2台持ちを薦めています。テザリングも出来るしね。Pocket-WiFiなんて要らんのです。

そして通信費にもっとストイックな貴方には、 Simフリー端末に データ通信のSimを指し、050plus を入れ、家の固定電話と携帯電話を解約すれば、かなりの節約生活ができますよ。実際電話をほぼ使わない私は、コレで充分な気がしてきています。

通信キャリアとしては、スマホを餌に高いパケット定額を売る気マンマンなんでしょうが、殆どの人にはオーバースペックだと思います。

もっと選択肢があってもいいと思うんですよね。

050plus が 、Simフリーと MVNO を普及させてくれると嬉しい。

なので私も使ってみて、使い勝手を知った上で、どんどん人に薦めていこうと思っています。

あとSo-net モバイル3G (これもDoCoMoのMVNO)でもイケるらしいです。

日本通信なら、b-mobile Fair（120日間または1GBまで 9,800円）か、b-mobile 1GB定額 (3,100円/月)ならイケるとの報告あり。DoCoMoのMVNOなんでエリアは問題ないでしょう。

あとSo-net モバイル3G (これもDoCoMoのMVNO)でもイケるらしいです。

ドコモのパケット定額や、各社MVNOなど、VoIPを制限しているもの多数だから注意が必要です。状況は変化するので、ググッて調べましょう。

 

 

登録が簡単/判りやすいのがイイ

登録はとっても簡単、必要なものはクレジットカード番号と、携帯電話（自動音声で本人確認がある）。

ウィザードにしたがって入力すれば、10分もかからない。

入会時には、友達紹介コードを入れましょう。

紹介したひと、された人双方に、500円分のQuoカードが抽選で当たるそうです。

参考サイトにいっぱい書いてますし、私のんも書いておきます。是非。

489GPA52

紹介相手には伝わらないようですので、気兼ねなく。

マニアな方々は、スマホ に SIPクライアントをぶっこんで どこでも IP電話を使える様にする御仁もおられるようですが、050plus はネームバリューにものを言わせ、それを安心・簡単にしてしまったのです。

この安心・簡単って、普及にはすごく大切な要素なんですよね。いくらSkypeでほぼ同じ事ができるとしても、面倒なんですよね。登録にPCが要るし、英語のページとか出てきた時点でアウトです。

 

 

050plus と Simフリー端末

個人的には、この050plus は、Simフリー端末でこそ威力を発揮するものだと思っています。

2012年以降はいよいよガラケーからスマホへのシフトが加速しそうですが、たかだか携帯電話ごときに月々6000円前後も払えるかい！と言う方が大半では無いでしょうか。

そんな人には、SimフリーのスマホをAmazonとかで購入して、DoCoMoの音声のみのプランで契約した携帯のSimを刺す（800円位/月）、というのを提案する様にしています。メールはGMailを使えばよいし、家と出先にWiFiがあればなんとかなります。急ぎの用事は音声でもSMSは使えるんで。各社乗り入れ可能になったいま、SMSってこういう場合にはアリかと。

いや外でもGMailとかネット見たいねん、と言う方には、これまたSimフリーのスマホにデータ通信プランのSimカードを指して、ガラケー（音声のみ契約）との2台持ちを薦めています。テザリングも出来るしね。Pocket-WiFiなんて要らんのです。

そして通信費にもっとストイックな貴方には、 Simフリー端末に データ通信のSimを指し、050plus を入れ、家の固定電話と携帯電話を解約すれば、かなりの節約生活ができますよ。実際電話をほぼ使わない私は、コレで充分な気がしてきています。

通信キャリアとしては、スマホを餌に高いパケット定額を売る気マンマンなんでしょうが、殆どの人にはオーバースペックだと思います。

もっと選択肢があってもいいと思うんですよね。

050plus が 、Simフリーと MVNO を普及させてくれると嬉しい。

なので私も使ってみて、使い勝手を知った上で、どんどん人に薦めていこうと思っています。
