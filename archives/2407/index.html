Macbook Air の SSD 容量不足と /.MobileBackups.trash

Macbook Air を使い始めて結構経ちますけど、128GBあれば余裕とか思っていたSSDがいつもいっぱいいっぱいです。

最初は調子乗って iPhotoやiTunesを使いまくったり XCode をインスコしたりしたせいだと思っていましたけど、どうもそれだけでは無い様です。

ターミナルから

sudo du -h -x -d 1 /

してみると、

/.MobileBackups.trash

なるファイルが41GBもありました。

なんじゃこりゃと思ってしらべると、Time Machine のオフライン用ジャーナルの様なものみたいです。

そういえば Time Machine で 外付けHDD にバックアップしてました。

という訳で、Time Machine を切 にすると消えました。
