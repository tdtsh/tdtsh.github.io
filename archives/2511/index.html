Windows版 eclipse が Failed to create The Java Virtual Machine で起動できない

ここんとこずっとMac OS X の eclipse しか触ってないんですけど、久しぶりにWindows XP で eclipse を起動しようとすると、エラーで起動出来なかった。

eclipse.ini にJREへのパスを追記することで回避できる。

-vm
C:\Program Files\Java\jre6\bin\javaw.exe

過去にも似たような事を書いているのにも関わらず、よく忘れるのでメモ。

それにしても、いつまでWindows XP を使うことになるんだろう自分。
