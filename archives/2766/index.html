Windows Azure のアカウント取ってみた

AWSみたいに楽ちんな IaaS を探しています。

root が取れるというか、CentOSが動いてSMTP送信させてくれたら結構なんでもいいです。

で、IaaSが追加された事は聞いてたんですが食わず嫌いだった Windows Azure のアカウント取って、CentOSとか動かしてみました。

とっても簡単だったのでメモる必要も無い気もしますが、一応スクショ撮ったので晒しておきます。

Microsoftアカウント(HotmailとかWindows Liveアカウント等)でのログインが必要です。

因みに私はMCSEとかMCDBAとか時代の頃から Hotmail も使っているんですが、最近 Windows 8を買ってからはGMailのアドレスでMicrosoftアカウントを作成しなおしたので、ソッチを使ってみました。

http://www.windowsazure.com/ja-jp/ から[無料で試す]をクリックする。(90日間の無料評価版)

→をクリック

携帯電話の電話番号を入力

SMSで送られてきた数字入れる

クレジットカード番号や個人情報をガッツリ入力する

使える様になりました

あとはコチラを参考に、CentOSとかの仮想サーバをたててしてみました。

登録開始からプロビジョニングが完了するまでの所要時間は10分位です。すごいですね。

WindowsAzureを使ってみる : メモ調

90日間は無料って事なんで、色々試したいと思います。
