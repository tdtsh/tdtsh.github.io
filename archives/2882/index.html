Mac で bootcamp 上の Windows 8 を VMWare に変更 (MSに電話)

経緯まとめ

愛機は Macbook Air Mid 2012

家のWindows機壊れる

オレ、これからは完全にマカーになる宣言

Officeとか必要。Mac 版Office使う

無理。やっぱWindows要る

Windows 8 Upgrade版を bootcamp で動かしてみる

ライセンス認証的に使い続けられない

Windows 8 64bit DSP版を買う

bootcamp で動かすも、キーボードやトラックパッドの動きが嫌になる

さらにMac OS Xと同時に Windows版Office使いたくなる

Virtual Box で古いWindows XP OEM版入れて動かす。

Virtual Box 無理。遅い。

VMware Fusion 買う。parallelsより安いし。

Bootcamp領域のWindows 8動かす。早い！

MSにライセンスで怒られる

Bootcamp領域をVMWareのVMとして書きだし、bootcamp領域削除

MSにライセンスで怒られ続けて困る← (イマココ)

 

電話でライセンス認証

仕方ないんで、Windows 8 のライセンス認証のガイダンスに従い、マイクロソフトに 電話しました。

音声ガイダンスで 7桁 x 8 位の数字を入力した後、オペレータに、Windows 8 64bit DSP版を bootcampやめて仮想領域だけで使う旨を説明し、 6桁 x 8位の数字を電話口で聞きながら入力すれば、ライセンス認証完了です。
