iOS用のインスペクタ Reveal をインストール

iOS用のインスペクタ、Reveal がいいかんじです。

iOSアプリのViewを作る時に。

今はベータ版という事で無料なんですけど、このままでは有料版買ってしまう勢いです。

 

インストール

http://revealapp.com/を開く

[Download BETA]をクリックして [Reveal.app.zip] をダウンロード

[Reveal.app.zip] を解凍し、アプリケーションにコピー

Revealを起動する

[Help]-[Show Reveal Library in Finder]を選択する

[Reveal.framework]のディレクトリをXcodeのプロジェクトにドラッグ&ドロップし、下記選択をして[Finish]をクリック

Copy items into destination group's folder (if needed) にチェック
    Create folder references for any added folers を選択

Xcodeプロジェクト内でTARGETS – プロジェクト名 – [Build Phases] を選択

[Link Binary with Language] に以下を追加(もしなければ)

CFNetwork.framework

QuartzCore.framework

Xcodeプロジェクト内でTARGETS – プロジェクト名 – [Build Settings]

[Other Linker Flags] に [-ObjC] を追加

ビルドしてシミュレータか実機でプロジェクトを実行

Revealを開く

左上の[No connection]をクリックすると、起動中の プロジェクト が選択できる
