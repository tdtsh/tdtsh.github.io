Windows 10 Mobile (Katana01) 触ってみた

TitaniumでWindows Mobileのアプリが作れる様になってから、実機欲しいなーっと思っていたので、1万円ちょいで買えると聞いて即買いしたFREETEL KATANA01が届いたので、初めてWindows Mobile触りました。

届いて、中身見て、はじめてデュアルSimであることを知りました。1こしか使わないけど。

とりあえず家にあったDocomoの回線に対し急遽moperaを追加して挿しました。ちなみにmoperaは申請してほぼ即使える様になりますので、MyDocomoでカジュアルにOFF/ONするで問題ないと思います。

筐体自体は、ピーチスキンな裏蓋でNexus5をやや縦長にしたような面持ちです。

嫌いじゃ無いけど、今となっては重く感じます。

将来子どもたちにお下がりになる気がするので、滑りにくいのは良いです。

1台目としてガッツリ使うのはやや厳し目だけど、電話とチャットくらいしか使わないなら及第点というか、なにしろ1万円ちょいなので買って後悔はしないと思います。

とりあえず入れてみたアプリ達

マシンが非力なので、メインで使うにはあまりにももっさりしていますが、とりあえず興味本位で、普段使いするなら真っ先に入れるであろうアプリを入れていみました。まぁ、正直苦行というか。Floyo以前のAndriodみたいな。フロンティアというかなんというか。

Dropbox -> 違和感ほぼ無い。さすが。

Evernote -> 一番使う写真撮影で紙とかを判定して切り抜くやつが使えない。

Google Drive -> 公式は無い。有料の野良アプリはあったけど入れてない。

Facebook -> 違和感無い。Android版とほぼ一緒。

Twitter -> クオリティはともかく、普通に使える。

Feedly -> ガワネイティブ。Google二段階認証の後またSignIn画面に。いつになったら使えるのか。

Pocket -> 無い。ガッカリ。

はてブ -> HBFavは勿論公式クライアントみあたらない。野良はてブリーダを1つ見つけて入れてみたけど、ほしいのはフォロワーの最新はてブなんでちょっと違う。

Slack(Beta) -> アカウント切り替えが出来ない。不便。

Messenger -> iOS/Android版と操作感の違和感少ないものの、なぜかフォントでかい

Chatwork -> 無い。非公式のが1つあったけどログイン出来ないので消した。

Line -> 「このアプリとデバイスには互換性がありません」だって

Skype -> 流石に出来が良い。違和感無い。

Outlookメール -> 標準MUA。GMailで使ってみた。意外と使える。件数多いラベルでの同期時間かかるし重い。

あと、MinecraftのPocket Editionがありました。

PC版、iOS版、Android版、Kindle版は購入済なので、コレクター的にも入れそうになったけど無駄遣い我慢。

スマホでExcelが使えるのは少しうれしいけど、iOS/Android版のGoogle Spreadsheetが優秀になってきた今となっては、特に必要性もいまいちな気が。

あと地味に不便なのが、日本語のユーザ定義辞書が無いことです。

仕事柄メールアドレスたくさん使い分けるのだけど、入力面倒だからユーザ辞書に登録して使いたいのだけど、Windows Mobileでは英語しか登録出来ないし、それさえも＠マーク入っていると登録できない。

個人的に、普段使いには致命的っす。

ちょっと今しごとたてこんでてアレなので、Tiでアプリ作るのとかは追々。
