AWS EC2 の CentOS でホスト名変更

全部ココに書いているけど、一部誤りがあるのでメモを残します。 Amazon EC2 Linux の静的ホスト名 RHEL7 Centos7

hostname変更

sudo vim /etc/hostname

yourhostname.your.doma.in

hosts変更

sudo vim /etc/hosts

127.0.0.1 yourhostname.your.doma.in localhost.localdomain localhost
::1       yourhostname.your.doma.in localhost.localdomain localhost6 localhost6.localdomain6

network変更

sudo vim /etc/sysconfig/network

HOSTNAME=yourhostname.your.doma.in

上書き防止

EC2起動時にIPアドレスベースのホスト名に自動で書き換えられるのを抑制

sudo vim /etc/cloud/cloud.cfg

preserve_hostname: true

あとはrebootすれば適用される
