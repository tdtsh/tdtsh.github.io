Google App Engine SDK for Java のインストール

1.]2 から、Google App Engine SDK for Java をダウンロードする。 2. ダウンロードしたSDKを、任意のフォルダに解凍する。※ここでは、c:\opt\java に解凍した。 3. コマンドラインで使うときの為に、環境変数を登録しておくと便利。

* SET PATH=%PATH%;c:\opt\java\appengine-java-sdk-1.2.6\bin * SET GAE_HOME=c:\opt\java\appengine-java-sdk-1.2.6 4. コマンドプロンプトを開き、サンプルアプリケーションを起動してみる

* cd %GAE_HOME% * bin\dev_appserver.cmd demos\guestbook\war
